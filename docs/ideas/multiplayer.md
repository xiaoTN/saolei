# 双人联机玩法灵感记录

> 创建日期：2026-03-10

## 候选模式

### 1. 抢地盘模式

- **核心规则**：共享一块大棋盘，双方各从一侧开始，翻开的格子归自己，最终谁占的地盘多谁赢
- **踩雷惩罚**：踩雷不结束游戏，可以扣分或冻结几秒作为惩罚
- **优点**：
  - 真正的对抗交互，有领地争夺的博弈
  - 中间区域的争夺会产生"要速度还是要安全"的策略抉择
- **待发散的问题**：
  - 踩雷的惩罚机制怎么设计？扣分 / 冻结 / 扣命？
  - 边界争夺区域如何避免变成纯赌运气？
  - 是否需要"视野迷雾"（只能看到自己已翻开区域附近的信息）？
  - 棋盘形状和大小怎么选？对称性很重要
  - 计分规则：纯格子数 vs 加权（雷附近的格子更值钱）？

### 2. 合作模式

- **核心规则**：共享棋盘，两人一起扫雷
- **生命机制**：每人有有限的生命数，踩雷扣命，全部耗尽则游戏失败
- **优点**：
  - 合作体验在扫雷中很新颖
  - 适合技术水平不同的玩家一起玩（带新手）
- **待发散的问题**：
  - 如何让"合作"有意义，而不是变成两人分区域各干各的？
  - 是否可以加入角色分工（比如一人翻格子、一人插旗）？
  - 是否引入"信息不对称" — 每人只能看到部分信息，必须配合？
  - 难度如何调整才能让合作有挑战感？
  - 沟通机制：语音 / 快捷指令 / 标记系统？
