# 💣 多边形扫雷

一个支持多种多边形棋盘的扫雷游戏，基于纯前端 SVG 渲染实现。

## ✨ 特性

- 🎮 **4种棋盘形状**：三角形、正方形、六边形、八边形+正方形混合
- 🎯 **5种难度**：简单、中等、困难、地狱、自定义
- 🖱️ **直观操作**：左键标记地雷，右键/长按打开格子
- ⚡ **首次点击安全**：第一次点击永远不会踩雷
- 📊 **实时状态**：格子数、雷占比、标记进度、计时器
- 📱 **响应式设计**：适配不同屏幕尺寸

## 🎮 玩法

| 操作 | 功能 |
|------|------|
| 左键点击 | 标记/取消标记地雷 🚩 |
| 右键点击 / 长按 | 打开格子 |

## 🚀 快速开始

### 在线游玩

直接用浏览器打开 `index.html` 即可开始游戏。

```bash
# 克隆仓库
git clone https://github.com/xiaoTN/saolei.git
cd saolei

# 方法1：直接打开
open index.html

# 方法2：使用本地服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

### 运行测试

```bash
# 安装依赖
npm install playwright

# 安装浏览器
npx playwright install chromium

# 运行测试
node test-game.js
```

## 📐 棋盘类型

### 三角形（3边）
- 平铺规律：尖朝上/朝下交替
- 每格 3 个邻居（共享顶点）
- 水平步进 cellSize/2

### 正方形（4边）
- 经典扫雷布局
- 每格最多 8 个邻居
- 对齐 Windows 经典标准

### 六边形（6边）
- 蜂窝状平铺
- 每格 6 个邻居
- 信息量略少于正方形

### 八边形+正方形（8边）
- 混合平铺：八边形与菱形交替
- 邻居数不固定
- 视觉效果独特

## 🎯 难度预设

| 边数 | 简单 | 中等 | 困难 | 地狱 |
|------|------|------|------|------|
| 3边 | 9×16/18雷 | 12×20/40雷 | 16×28/100雷 | 100×100/2500雷 |
| 4边 | 9×9/10雷 | 16×16/40雷 | 16×30/99雷 | 100×100/2500雷 |
| 6边 | 8×8/10雷 | 11×13/30雷 | 14×18/60雷 | 100×100/2500雷 |
| 8边 | 5×6/10雷 | 7×8/28雷 | 9×10/55雷 | 100×100/2500雷 |

> 难度参考 Windows 经典扫雷标准

## 📝 开发笔记

### 核心算法

1. **多边形邻居计算**：基于几何关系计算每个格子的邻居
2. **首次点击安全**：第一次点击后放置地雷，避免踩雷
3. **自动展开**：空白区域递归展开
4. **O(1) 胜利判断**：通过计数器实现高效判定

### 性能优化

- 邻居缓存
- 格子列表缓存
- SVG 复用
